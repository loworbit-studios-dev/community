# Vale of Ruin — Bug Reports

This repository is the official bug tracker for **Vale of Ruin**, a dark medieval strategy MMO developed by [Low Orbit Studios](https://loworbit-studios-dev.github.io).

---

## Reporting a Bug

Use [GitHub Issues](https://github.com/loworbit-studios-dev/community/issues) to submit bug reports.

**Before submitting, please:**
- Search existing issues to check your bug hasn't already been reported
- If it has, add a comment with any additional detail — duplicate reports with extra information are still useful

**When submitting, please include:**
- What happened
- What you expected to happen
- Steps to reproduce the issue
- Your device and OS version
- Your realm name (once the game is live)
- Screenshots or recordings if possible

Every report is read, triaged, and tracked. Swift resolution is a commitment, not a promise.
